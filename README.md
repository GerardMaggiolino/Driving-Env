# gym-driving
Eventually an OpenAI gym environment. 

### To install: 

When in gym-driving directory, 
```bash
pip install -e . 
```

### To run: 

```python
import gym 
import gym_driving

env = gym.make('driving-v0')
```

### To edit: 

Edit class DrivingEnv in gym-driving/gym_driving/envs/driving_env.py
